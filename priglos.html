🔧 Инструкция по созданию Twitch Extension

Шаг 1: Создание расширения в Developer Console

  1. Перейдите на Twitch Developer Console
  2. Нажмите "Your Extensions" → "Create Extension"
  3. Выберите тип расширения: Panel
  4. Заполните базовую информацию о расширении

Шаг 2: Настройка расширения

  1. В разделе "Extension Capabilities" включите:
    • Chat Capabilities
    • Broadcast Capabilities
    • Authentication Capabilities
  2. В разделе "Extension Views" добавьте:
    • Panel: https://your-domain.com/extension.html
    • Broadcast Configuration: https://your-domain.com/config.html

Шаг 3: Ключи и настройки

// Вам понадобятся эти ключи:
Client ID: (автоматически генерируется)
Secret Key: (будет показан один раз)

// Важные настройки:
- Все URL должны быть HTTPS
- Домен должен быть верифицирован
- Расширение должно пройти модерацию

Шаг 4: Загрузка кода

  1. Создайте файлы: extension.html, config.html
  2. Загрузите на ваш HTTPS сервер
  3. Укажите URLs в настройках расширения
  4. Отправьте на модерацию
